## ⚡ Overview

Shotgun Ninjas is a modular digital platform designed to host and scale multiple creative and technical products under one unified brand.

This repository powers the core website and application hub, including:

- 🎵 **Sound Studio** – Original music, beats, and audio experiences
- 🧠 **Ninjamation** – AI-powered creative and automation tools
- 🎮 **Labyrinth Ronin** – Survival-based game experience
- 🧰 **Arsenal** – Internal and public tools, utilities, and experiments

The platform is built with performance, scalability, and visual identity in mind.

---

## 🧱 Tech Stack

- **Frontend:** React + TypeScript
- **Build Tool:** Vite
- **Styling:** Tailwind CSS + shadcn/ui
- **UI Philosophy:** Dark, cinematic, component-driven
- **Deployment:** Lovable (with GitHub integration)
